# Friendly Android

> Make new friends!

Private social network to make meeting friends easier. You can learn more on our page: [getfriend.ly](https://github.com/friendly-social/android).

## Download

Download APK from direct link: [friendly-latest.apk](https://github.com/friendly-social/android/releases/download/nightly/friendly-latest.apk).

This APK is built on GitHub Actions from each commit and is pushed to [Nightly Release](https://github.com/friendly-social/android/releases/tag/nightly).
